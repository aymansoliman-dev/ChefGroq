# Chef Groq

**Chef Groq** is an AI-powered recipe generator that helps you turn any ingredients you have on hand into creative, delicious dishes ‚Äî instantly.
Built with **React**, **Tailwind CSS**, and **Vite** for a fast, modern web experience.

---

## Features

* AI-generated recipes based on your ingredients
* Clean, responsive UI with Tailwind CSS
* Instant rendering and smooth animations
* Mobile-friendly design
* Minimal, distraction-free interface

---

## Tech Stack

| Technology         | Description                               |
| ------------------ | ----------------------------------------- |
| **React**          | Front-end library for UI components       |
| **Vite**           | Lightning-fast development and build tool |
| **Tailwind CSS**   | Utility-first CSS framework               |
| **React Markdown** | Render recipe text beautifully            |
| **Vercel**         | Deployment and hosting                    |

This app is deployed on **Vercel**.
You can easily deploy your own by connecting this repo and setting the build command to:

```bash
npm run build
```

and the output directory to:

```bash
dist
```
